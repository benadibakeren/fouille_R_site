# Mini-projets (1, 2, 3)

Les mini-projets appliquent les notions de fouille de données à des **datasets
textuels**. Chaque projet suit un pipeline classique : nettoyage, vectorisation
TF-IDF, réduction/sélection, classification, évaluation.
